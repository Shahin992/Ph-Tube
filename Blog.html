আন্ডিফাইন্ড: আন্ডিফাইন্ড মানে একটি ভেরিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু এখনও এর কোন মান দেওয়া হয়নি। ভেরিয়েবলটি ডিক্লেয়ার করা হলে মান নির্ধারণ করা না হওয়ায় আন্ডিফাইন্ড হয়। ফাংশন প্যারামিটার যেগুলোর কোন মান দেওয়া না হলে তা ডিফল্ট মান আন্ডিফাইন্ড হতে পারে। অবজেক্ট প্রপার্টি বা অ্যারে এলিমেন্টে যখন কোনও মান নেই সেটা আন্ডিফাইন্ড হতে পারে। একটি ভেরিয়েবল একটি ফাংশন থেকে return স্টেটমেন্ট ছাড়া ফিরে আসলে সেটি ডিফল্টভাবে আন্ডিফাইন্ড হয়।

What do you mean by REST API?

REST API একটি প্রোগ্রামিং ইন্টারফেস হলো, যা Representational State Transfer (REST) নামক একটি ডিজাইন ফলে তৈরি হয়। এটি অধিকাংশ ওয়েব সার্ভিসের সাথে সম্পর্কিত কাজ করে এবং ওয়েব অ্যাপ্লিকেশনের সম্প্রতি প্রয়োজনীয়তা হিসেবে ব্যবহৃত হয়। REST API এর মৌলিক ধারণা হলো ওয়েব রিসোর্সের (ডেটা বা সার্ভিস) প্রতি একটি স্থিতি প্রতিষ্ঠাপন করা, যেটি স্থিতি প্রতিষ্ঠাপন এবং ব্যবহারের জন্য HTTP প্রতিবেদন (HTTP requests) ব্যবহার করে অ্যাক্সেস করা হয়। প্রতিটি REST API রিসোর্স একটি ইউনিক ইউআরআই (Uniform Resource Identifier) দ্বারা সনাক্ত করা হয়, এবং HTTP প্রতিবেদনগুলি সাধারণভাবে ডেটা প্রতিস্থাপন এবং প্রাপ্ত করার জন্য ব্যবহৃত হয়। REST API-এ চারটি মৌলিক HTTP প্রতিবেদন প্রয়োজন:

GET: সার্ভার থেকে ডেটা পড়ে আনার জন্য ব্যবহৃত হয়।
POST: ডেটা সার্ভারে প্রেরণ করতে ব্যবহৃত হয়।
PUT: সার্ভারে ডেটা আপডেট করতে ব্যবহৃত হয়।
DELETE: ডেটা DELETE ফেলতে ব্যবহৃত হয়।
